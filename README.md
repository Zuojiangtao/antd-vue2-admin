# antd-vue2-admin

## Project setup

### Clone repo

```bash
git clone --depth=1 https://github.com/Zuojiangtao/antd-vue2-admin.git my-project
cd my-project
```

### Install dependencies

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
